Technologies Covered in Training

Git

Git is basically a version control system. It is DVCS which means Distributed Version Control system. Git is used for tracking changes in source code during software development. It is designed for coordinating work among programmers, but it can be used to track changes in any set of files.

GitHub

GitHub is a Git repository hosting service, but it adds many of its own features. While Git is a command line tool, GitHub provides a Web-based graphical interface. It also provides access control and several collaboration features, such as wikis and basic task management tools for every project.

MySQL

MySQL is a freely available open source Relational Database Management System(RDBMS) that uses Structured Query Language(SQL). SQL is the most popular language for adding accessing and managing content in a database. It is most noted for its quick processing, proven reliability,ease and flexibility of use.

Django

Django is a free and open-source web application framework written in python. A framework is nothing more than a collection of modules that make development easier. This framework is also used as backend development for a web application. This integrates the frontend part of the website and make websites more user friendly.

Flask

Flask is a microframework written in Python. It is classified as a microframework because it does not require particular tools or libraries. It has no database abstraction layer, form validation, or any other components where pre-exisiting third-party libraries provide common features.

Android Studio

Android Studio is the official integrated development environment(IDE) for Google's Android operating system.
